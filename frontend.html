Frontend

¿que es frontend?

Frontend o desarrollo del lado del cliente, es la parte de un programa o dispositivo a la que el usuario puede acceder directamente, son las tecnologias de diseño y desarrollo web que correne en el navegador se encargan de la interactividad con los usuarios.

Fundamentos de frontend

El desarrollador front-end trabaja con un diseñador grafico que proporciona el diseño grafico, el desarrollador front-end trabaja con un disñador UX que propone y crea un prototipo. Los desarrolladores usan JavaScritp que agrega las interacciones Tiene que tener conocimiento en HTML + CSS ya que son basicos para construir un sitio web. jQuery, es una biblioteca de JavaScritp que permite escribir JavaScritp mas facil y rapido.
